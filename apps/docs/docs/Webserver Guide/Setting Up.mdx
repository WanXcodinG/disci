# Setting Up

Once you have installed the framework, you can create all our base files.

Start by Running the following command with the package manager of your choice to create the package.json file

<Tabs groupId="packageManager">
  <TabItem value="npm" label="Npm" default>
     <CodeBlock className="language-bash">
        npm init -y
     </CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    <CodeBlock className="language-bash">
        yarn init
     </CodeBlock>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
     <CodeBlock className="language-bash">
        pnpm init
     </CodeBlock>
  </TabItem>
</Tabs>
